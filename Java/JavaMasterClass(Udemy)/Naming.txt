<패키지>
항상 소문자
유일해야한다
인터넷 도메인주소를 거꾸로

<도메인주소>
- 는 _로변경
숫자로 시작하는 도메인주소는 _로 시작
자바 키워드로 시작하는 도메인 주소는 _로시작

ex1) Switch.supplier.com -> com.supplier_switch
ex2) 1world.com -> com._1world
ex3) Experts-exchange.com -> com.experts_exchange


<클래스명,인터페이스>
CamelCase
명사형이어야한다.
대문자로 시작해야한다.
*인터페이스 :Comparable,Serializeable

ex1) ArrayList
ex2)LinkedList
ex3)String
ex4)TopSong
ex5)GearBox


<메소드명>
동사형 
대소문자 섞임

ex1)size()
ex2)getName()
ex3)addPlayer()


<상수명>
전부 대문자
_ 로 띄어쓰기 대체

<변수명>
대소문자 섞어씀
 소문자로 시작
_ 쓰지않음

<타입변수>

대문자 한글자만 사용

E-element,K-key,T-type,V-value
S -second type,U-third type,V-fourth type
