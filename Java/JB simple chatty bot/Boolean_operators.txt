OR is a binary operator that returns true if at least one operand is true, otherwise, it returns false.
boolean b3 = true || false;  // true
boolean b4 = true || true;   // true

XOR (exclusive OR) is a binary operator that returns true if boolean operands have different values, otherwise, it is false.
boolean b3 = true ^ false;  // true
boolean b4 = true ^ true;   // false

&&: 하나라도 false면 false
||: 하나라도 true면 true
^: 둘이 다르면 true
